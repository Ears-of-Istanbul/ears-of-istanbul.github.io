Our Project

Ears of Istanbul

Noise pollution is a big problem for the cities. Sometimes citizens may ignore this problem or not understand it at all. It may take a long time for the relevant institutions to dynamically respond to this problem. In such cases, noise pollution can turn into the biggest problem in our ears.

If the citizens read and understand the data more easily, we can create a more conscious society. Relevant institutions seeing of the problem locations boosts the solution process too.
We (as the EoI Team) aim to create a more conscious society with using a dynamic sound map of our city. With the decibel meter, gps module and internet module in Ear Box -which you can call tiny computers- we can see the location of the Ear Box and the decibels it measures in the internet. We create an easy-to-read and dynamic sound maps with visualizing our data.

In the near future, we want to be more developable system so we plan to open source our project. Maybe our project is the beginning of the a dynamic sound map of the world.
